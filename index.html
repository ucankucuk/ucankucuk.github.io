Java StringBuilder & StringBuffer

  • StringBuilder nesneleri değiştirilebilir olması haricinde String nesnelerine benzer.
  • StringBuilder karakter dizisi içeren değişken uzunlukta ki diziler gibi davranır.
  • Herhangi bir noktada metod çağırımıyla uzunluk ve dizi içeriği değiştirilebilir.
  • Örneğin StringBuilder nesnelerinde append() , insert(), reverse() , replace() metodları vardır.
  • Çok sayıda dizileri bağlamada , StringBuilder nesnelerini kullanmak daha verimlidir.

Length & Capacity

  • StingBuilder sınıfı length() metoduna sahiptir.Bu özelliğiyle String sınıfına benzerdir.Ayrıca StringBuilder'ın capacity() metodu vardır.Bu metod StringBuilder'ın kapasitesini geri döndürür.Bu özellik String sınıfında yoktur.
Yapıcı(Constructor) Açıklama
StringBuilder() Boş bir StringBuilder oluşturur(Kapasitesi : 16)
StringBuilder(CharSequence cs) CharSequence içeriğinin aynısını 16 boş elemanlı haliyle bir
StringBuilder oluştururur.
StringBuilder(int initCapacity) initCapacity uzunluğu kadar kapasiteli bir StringBuilder
oluşturur.

  • StringBuilder sınıfı setLength(int newLength) ile karakter dizisinin uzunluğunu ayarlarken newlength değeri kendi uzunluğundan büyük olursa dizinin sonuna eksik kadar null değeri ekler.Eğer newlength değeri kendi uzunluğundan kısaysa da dizinin sonundan gerekilen kadar karakteri siler.
package ucankucuk.github.io;

public class SetLengthMetodExample {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder(); // 16 kapasiteli StringBuilder nesnesi oluşturuldu.
        System.out.println("Boş StringBuilder nesnesini kapasitesi = " + sb.capacity());
        System.out.println("Boş StringBuilder nesnesinin uzunluğu = " + sb.length());
        sb.append("0123456789ABCDEF");
        System.out.println("Eklenen = 0123456789ABCDEF" + " Eklemeden sonra uzunluk = " + sb.length());
        sb.append("UTKU");
        System.out.println("StringBuilder nesnesini içeriği = " + sb);
        System.out.println("Son eklemeden sonra uzunluk = " + sb.length());
        System.out.println("Son eklemeden sonra kapasite = " + sb.capacity());

        /*
         * 16 kapasitesindeki sb nesnesine 4 karakter daha eklediğimizde hata vermedi.
         * Çünkü StringBuilder otomatik olarak kapsiteyi arttırdı.
         */
    }

}

  • Ekran Çıktısı:

    • Boş StringBuilder nesnesini kapasitesi = 16
    • Boş StringBuilder nesnesinin uzunluğu = 0
    • Eklenen = 0123456789ABCDEF Eklemeden sonra uzunluk = 16
    • StringBuilder nesnesini içeriği = 0123456789ABCDEFUTKU
    • Son eklemeden sonra uzunluk = 20
    • Son eklemeden sonra kapasite =34

  • Ayrıca append(), insert(), setLength() metodları çağırıldığından kapasite otomatik olarak büyür.

package ucankucuk.github.io;

public class IncreasedAutomatically {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder();
        sb.append("0123456789ABCDEF");
        System.out.println("StringBuilding uzunluğu = " + sb.length() + " StringBuilder kapasitesi = " + sb.capacity());
        sb.append("UTKU");
        sb.setLength(5);
        System.out
                .println("Değişiklikten sonraki StringBuilder nesnesi = " + sb + "\n" + "Değişiklikten sonraki kapasite = " + sb.capacity() + "\n" + "Değişiklikten sonraki uzunluk = " + sb.length());
        /* insert() metodu ile belirlenen yere ekleme yapabiliriz. */
        sb.insert(5, "5");
        System.out.println("Eklemeden sonra StringBuilder nesnesi = " + sb + "\n" + "Eklemeden sonraki kapasite = " + sb.capacity() + "\n" + "Eklemeden sonraki uzunluk = " + sb.length());
    }
}
